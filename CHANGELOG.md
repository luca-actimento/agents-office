# Changelog

## v1.0.0

Initial release of Agents Office.

### Features

- **Animated AI agent characters** — Each Claude Code terminal gets its own pixel-art character that moves, types, and reads in real time
- **Live activity tracking** — Characters react to what the agent is doing (writing code, reading files, searching, running commands)
- **Office layout editor** — Full tilemap editor with floor painting, wall placement, and furniture
- **100+ furniture assets** — Desks, chairs, bookshelves, electronics, wall decor, and more
- **Drag & Drop furniture** — Rearrange your office without entering edit mode
- **Furniture catalog panel** — Browse, search, and place items from a built-in catalog
- **Cafe zone** — Idle agents visit the office cafe for a coffee break with custom pathfinding
- **Door system** — Auto-opening doors with sound effects when agents walk through
- **Permission blink** — Red blinking overlay when an agent needs your approval
- **Sub-agent visualization** — Task sub-agents get their own characters
- **Seat assignment** — Click a character, then click a seat to reassign
- **Sound notifications** — Audio chime when an agent finishes or needs attention
- **Undo/Redo** — 50-level undo in the layout editor
- **Layout export/import** — Share office layouts as JSON
- **Furniture rotation** — Right-click to rotate furniture
- **Furniture actions** — Double-click furniture to trigger custom commands
- **Auto-updates** — One-click update notifications from GitHub releases
